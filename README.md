# Daily Thinking Challenge

A personal self-improvement web app that shows one structured thinking challenge per day — to sharpen problem-solving, creativity, and mental agility.

Built for myself to develop the kind of thinking that PMs, founders, and consultants rely on daily. One challenge. One day. No skipping.

We use the following tech stack:
- Google Gemini for generating and personalizing daily challenges
- TypeScript + React for the interface
- Tailwind CSS for styling
- localStorage for streak and history tracking

## 🚀 Features

* **Daily Challenge:** A fresh structured problem every 24 hours — forces a daily practice habit
* **Challenge Categories:** Logical reasoning, creative thinking, decision-making, mental math, problem decomposition
* **Reflection Space:** Write out your thinking before the solution is revealed — the most important step
* **Streak Tracker:** Tracks consecutive daily completions to build and maintain momentum
* **Insight Library:** Review past challenges and your own responses to see how your thinking has evolved

## Setup

```bash
git clone https://github.com/yatinbhalla/Daily-Thinking-Challenge.git
cd Daily-Thinking-Challenge
npm install
echo "GEMINI_API_KEY=your_key_here" > .env.local
npm run dev
```

Get a free Gemini API key at [aistudio.google.com](https://aistudio.google.com/)
